5. Авто-сохранение (ide.txt пункты 18-19)

Проблема: В AutoSaveManager.cpp логика не полностью соответствует требованиям.

``` cpp
// Требование пункт 19:
// "Если файл не редактировался более 5 минут, его копия
// однократно сохраняется... и больше автосохранений не происходит"

// Реализация в AutoSaveManager.cpp строка ~250:
bool AutoSaveManager::shouldSave(const AutoSaveEntry& entry) const {
    if (!entry.isDirty) return false;

    // Проблема: не отслеживается флаг "уже сохранён после 5 минут бездействия"
    if (entry.isAutosaved) {
        return false;  // ✅ Правильно
    }

    // Проблема: проверка времени бездействия некорректна
    qint64 minutesSinceEdit = entry.lastEdit.secsTo(now) / 60;
    if (minutesSinceEdit >= intervalMinutes) {
        return true;  // ❌ Не проверяет, было ли уже однократное сохранение
    }

    return false;
}
```

Требуется: Добавить флаг singleSavePerformed для отслеживания однократного сохранения.

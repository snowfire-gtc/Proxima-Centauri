🚀 Запуск тестов:

``` bash
# Все тесты
./run-tests.sh all

# Конкретная категория
./run-tests.sh compiler
./run-tests.sh ide
./run-tests.sh language
./run-tests.sh performance
./run-tests.sh collaboration

# С очисткой
./run-tests.sh -c all

# Подробный вывод
./run-tests.sh --verbose all
```

Система интеграционного тестирования содержит полный набор тестов для Proxima Compiler & Centauri IDE!

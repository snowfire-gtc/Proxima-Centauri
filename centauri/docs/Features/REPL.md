## Реализованные команды REPL:

✅ help - справка по командам
✅ whois - детальная информация о переменной
✅ whos - список всех переменных
✅ clear - очистка консоли/переменных/истории
✅ history - история команд
✅ load - загрузка скрипта
✅ save - сохранение workspace
✅ type - тип переменной
✅ size - размер переменной
✅ class - класс переменной
✅ methods - методы класса
✅ fields - поля класса
✅ exit - выход из REPL
✅ version - версия
✅ path - пути поиска
✅ cd - смена директории
✅ pwd - текущая директория
✅ ls - список файлов
✅ edit - открыть в редакторе
✅ run - запуск скрипта
✅ debug - отладка скрипта
✅ memory - использование памяти
✅ config - конфигурация REPL
✅ reset - сброс состояния

## Функции:

✅ Вкладка в консоли - отдельная вкладка REPL
✅ История команд - с сохранением между сессиями
✅ Автодополнение - команд и переменных
✅ Подсветка синтаксиса - цветовая
✅ Управление переменными - whois, whos
✅ Загрузка скриптов - load команда
✅ Сохранение workspace - save команда
✅ Интеграция с Runtime - выполнение кода
✅ Интеграция с TypeChecker - информация о типах
✅ Настройка prompt - конфигурация

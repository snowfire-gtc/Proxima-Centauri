## Реализованные функции:

✅ 3 режима рендеринга - CPU, GPU OpenGL, GPU CUDA
✅ 7 режимов отображения - Volume, Slice XY/XZ/YZ, Wireframe, Points, Surfaces
✅ 1-based индексация - согласно language.txt пункт 11.1
✅ Интерактивная камера - вращение, панорамирование, zoom
✅ Прозрачность вокселей - настраиваемый порог
✅ 17 цветовых палитр - Viridis, Plasma, Inferno, etc.
✅ Выделение вокселей - с tooltip
✅ Срезы - XY, XZ, YZ плоскости
✅ Статистика - min, max, mean, visible count, FPS
✅ Экспорт/Импорт - PNG, JPG, CSV
✅ Печать - через QPrinter
✅ Контекстное меню - все операции
✅ Горячие клавиши - R (reset), Ctrl+S (save), etc.
✅ Редактирование - изменение значений вокселей
✅ High Performance Mode - для больших объёмов данных
✅ GPU детекция - автоматический выбор режима
✅ CUDA поддержка - при наличии

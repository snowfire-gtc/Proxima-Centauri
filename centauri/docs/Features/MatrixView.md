## Реализованные функции:

✅ 3 режима отображения - Table, Image, Heatmap
✅ 17 цветовых палитр - Viridis, Plasma, Inferno, Magma, etc.
✅ 1-based индексация - согласно language.txt пункт 11.1
✅ Zoom/Pan - масштабирование и перемещение
✅ Выделение диапазона - с копированием данных
✅ Наложения - матрицы, графики, точки
✅ Отображение значений - при достаточном zoom
✅ Заголовки - строк и столбцов
✅ Обрезка матрицы - по выделению
✅ Экспорт/Импорт - PNG, JPG, CSV
✅ Печать - через QPrinter
✅ Статистика - min, max, mean, sum
✅ Контекстное меню - все операции
✅ Горячие клавиши - Ctrl+C, Ctrl+V, Ctrl+A, etc.
✅ Интерактивность - редактирование ячеек
✅ Drag-and-drop - загрузка CSV
✅ Tooltip - значение при наведении

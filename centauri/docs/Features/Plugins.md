## Поддержка плагинов:

✅ Загрузка плагинов - динамическая загрузка через QPluginLoader
✅ Выгрузка плагинов - корректное освобождение ресурсов
✅ Сканирование директорий - автоматический поиск плагинов
✅ Валидация - проверка подписей и метаданных
✅ Расширения - система extension points
✅ Настройки - сохранение/загрузка конфигурации
✅ Установка - загрузка плагинов из сети
✅ Обновление - проверка и установка обновлений
✅ Удаление - корректное удаление плагинов
✅ Зависимости - проверка зависимостей между плагинами
✅ Платформы - проверка совместимости с ОС
✅ Таймауты - защита от зависания при загрузке
✅ Прогресс - отображение прогресса сканирования
✅ Логирование - полная интеграция с Logger

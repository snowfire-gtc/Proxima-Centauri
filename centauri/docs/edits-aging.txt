Режим "старость" показывает не старость автора, а сколько прошло времени с момента внесения правки в блок текста. Для этого, предлагаю в папке ".proxima/story", для каждого исходного текста, например, для "abc.prx", хранить в файле ".proxima/story/abc.story" историю правок с указанием даты правки в формате collection: [<время изменения>, <add/remove>, <текст правки>], например: добавление ["13/05/2025 13:01",  "add", "a: single = 0.0;"], удаление: ["23/06/2025 03:44", "remove", "a: single = 0.0;"], при этом запись делается для каждого блока текста, изменение которого можно отменить (или вернуть) за одно нажатие кнопки отмены последней правки. При включении режима "старости", текующее состояние файла, например, "abc.prx" сравнивается с содержимым соответствующего файла ".proxima/story/abc.story" истории и подсветкой отображаются цвета, соответствующие времени, которое прошло с момента внесения правки.

# Горячие клавиши сворачивания кода

## Основные операции

| Действие | Windows/Linux | macOS |
|----------|---------------|-------|
| Свернуть/развернуть текущий блок | `Ctrl+-` | `Cmd+-` |
| Развернуть текущий блок | `Ctrl++` | `Cmd++` |
| Свернуть все блоки | `Ctrl+Shift+0` | `Cmd+Shift+0` |
| Развернуть все блоки | `Ctrl+Shift+1` | `Cmd+Shift+1` |
| Свернуть уровень 1 | `Ctrl+Shift+2` | `Cmd+Shift+2` |
| Свернуть уровень 2 | `Ctrl+Shift+3` | `Cmd+Shift+3` |

## Типы сворачивания

| Действие | Горячие клавиши |
|----------|-----------------|
| Свернуть методы | `Ctrl+M` |
| Свернуть классы | `Ctrl+K` |
| Свернуть регионы | `Ctrl+R` |
| Свернуть комментарии | `Ctrl+C` |

## Навигация

| Действие | Горячие клавиши |
|----------|-----------------|
| Перейти к следующему сворачиваемому блоку | `F2` |
| Перейти к предыдущему сворачиваемому блоку | `Shift+F2` |

## Контекстное меню

Правый клик на маркере сворачивания:
- Свернуть
- Развернуть
- Свернуть все на этом уровне
- Развернуть все на этом уровне
